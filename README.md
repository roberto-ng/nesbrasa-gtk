# Nesbrasa 
## Emulador de NES e Famicom para Windows e GNU/Linux

Repisitório da biblioteca núcleo: https://gitlab.com/robertonazareth/nesbrasa

### Dependências:
* Compilador para C++17
* Meson Build System
* Ninja
* Gtkmm >= 3.24.1

### Instruções para compilação:

```
meson _build
cd _build
ninja
```
